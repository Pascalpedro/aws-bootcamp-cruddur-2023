# PYTHON & IT'S ENVIRON TOOLS

## Install python version

```py
pyenv install 3.10.9
```

## Set your python version

```py
pyenv global 3.10.9
```

## Create virual environment

```py
python -m venv venv
```

## Activate environment

```py
source venv/bin/activate
```

## Install Flask

```py
pip install flask
```
